# KOL 双语沟通助手 · 团队安装与更新指南

> 给团队成员看的。装一次就行，以后话术和 AI 更新都是自动的，你什么都不用做。

---

## 一、第一次安装（每人做一次，约 3 分钟）

1. 拿到管理员发的插件压缩包 `kol-bilingual-assistant.zip`，**解压**到一个固定文件夹（比如桌面，别留在压缩包里）。
2. 打开 Chrome / Edge，地址栏输入 `chrome://extensions`（Edge 用 `edge://extensions`）回车。
3. 打开右上角的「**开发者模式**」。
4. 点「**加载已解压的扩展程序**」，选中刚才解压出来的文件夹（里面能直接看到 `manifest.json` 那一层）。
5. 点浏览器工具栏的插件图标，打开右侧边栏，点顶部「**⚙️ 服务器设置**」：
   - 服务地址：**已经默认填好了**，无需修改。
   - 团队口令：填**管理员发给你的口令**。
   - 管理员口令：**留空**（普通成员不用填）。
   - 点「**保存并连接**」，右上角显示「**千问已连接**」就成功了。

完成！打开 Instagram 就能用了：红人外语消息会自动翻译，点插件图标开侧边栏可以分析、回复、选话术。

## 二、以后怎么更新？

分两种，绝大多数时候你**什么都不用做**：

| 更新内容 | 你要做什么 |
|---|---|
| **话术库**（新增/修改话术） | ✅ 自动，无需操作（话术存在服务器，实时同步） |
| **AI 逻辑 / 翻译效果** | ✅ 自动，无需操作 |
| **插件界面 / 新功能**（很少） | 管理员发新压缩包 → 解压后**覆盖进原来那个文件夹**（同一位置、别删插件）→ `chrome://extensions` 点插件卡片上的 **🔄 刷新** 即可 |

> 简单说：平时话术怎么变都不用管；只有界面有大改时，把新文件覆盖进**原文件夹**再点刷新。
> ⚠️ **别**「下载到新位置 → 删旧的 → 加载新文件夹」，那样路径变了会丢本地记录（合作进度/提醒/待办）。
> 从 0.22.1 起插件 ID 已固定，丢失风险已堵住；⚙️ 服务器设置里还有「💾导出备份 / 📂导入恢复」可换电脑搬家 / 还原。

## 三、常见问题

- **右上角显示「AI 未启动 / 待配置」？** 打开「⚙️ 服务器设置」确认团队口令填对了，点「保存并连接」。
- **看不到「选话术」里的话术？** 确认已「千问已连接」；换个产品或清空搜索再看。
- **关掉浏览器后还在吗？** 在，装一次就一直在，除非你手动移除。
