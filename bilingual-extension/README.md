# KOL 双语沟通助手（原型）

这是一个 Chrome / Edge Manifest V3 插件，优先用于 Instagram 网页版，也支持 Gmail 和 Outlook 网页版。AI 使用阿里云百炼千问，通过本机服务调用，API Key 不会写入插件。

## 当前可用功能

- 在 Instagram、Gmail 或 Outlook 页面显示“译”悬浮按钮
- 读取网页中选中的红人消息
- 用 AI 翻译、判断真实意图并匹配话术
- Instagram 私信中的外语消息会自动在原消息下方显示中文翻译
- 分开显示字面意思和可能的言外之意
- 将外发回复与内部处理建议严格隔离
- 标记是否需要询问 TL，并生成可复制给 TL 的问题
- 等待 TL 确认时生成安全的临时回复
- 支持选择不同产品资料包
- 支持在侧边栏直接问千问，用于追问、纠错和重新理解原文
- 支持将确认后的分析保存为本地话术场景
- 外语回复和中文回复都可以直接编辑
- 中文框可输入完整回复，也可只写大概意图，由千问生成并润色双语终稿
- 修改外语回复后可一键重新翻译中文用于校对
- 提供主动话术快捷库，可选择场景、语言并一键生成
- 已确认存档会优先参与后续话术匹配
- 支持搜索存档和导出 JSON 备份
- 展示外语回复、中文对照与风险提醒
- 支持补充上下文和运营回复目标
- 一键复制回复
- 没启动 AI 服务时自动降级到离线话术

## 安装

1. 打开 Chrome 的 `chrome://extensions` 或 Edge 的 `edge://extensions`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目文件夹
5. 双击插件目录中的 `reconfigure-assistant.cmd`，首次输入阿里云百炼 API Key
6. 配置成功后关闭窗口
7. 双击 `install-background-startup.cmd`
8. 安装成功后，AI 服务会在后台静默运行，并在登录 Windows 时自动启动

## 更新插件

文件更新后不需要重新添加扩展，只需进入扩展管理页面点击“刷新”。

## 后台服务

- `install-background-startup.cmd`：安装开机自启并立即在后台启动
- `stop-assistant.cmd`：停止当前后台服务
- `uninstall-background-startup.cmd`：取消开机自启并停止服务
- `reconfigure-assistant.cmd`：更换百炼 API Key
- `data/assistant-output.log`：后台服务正常输出
- `data/assistant-error.log`：后台服务错误日志
- `data/assistant-status.log`：启动状态记录

## 翻译与分析的区别

- 消息下方翻译：只做忠实翻译，严格保留谁让谁做什么，不进行业务推测
- 侧边栏分析：判断言外之意、匹配场景、建议询问 TL 和生成回复
- 直接问千问：当分析不准确时，可针对原文继续追问或要求纠正

## 产品资料

- 产品资料保存在 `data/products.json`
- 当前包含“通用产品”和 Recco 示例
- 新增产品时复制一条产品记录，并填写产品名、简介、社媒账号、下载方式、卖点、Brief、授权规则及禁止承诺事项

## 话术存档

- 分析完成后可先修改外语和中文回复，再点击“保存为确认话术”
- 填写清晰的场景名称和运营备注
- 存档记录产品、触发消息、正确理解、最终编辑后的外发回复、内部建议、变量和版本号
- 数据保存在 `data/scenario-archive.json`
- 建议定期点击“导出备份”

## 主动话术快捷库

内置场景包括：

- 冷启动私信、冷启动邮件
- 询问报价、档期和缺失条件
- 催回复、催初稿、催修改稿、催发布
- 二次合作、多条合作、长期合作
- 从公司其他产品转来询问新合作
- 拉群后第一句、合作流程和发送合同
- 发送 Brief、要求修改
- 审核通过、申请付款和付款完成

没有变量的场景点击后直接生成；需要产品名、数量、日期、链接等信息时会先显示填写框。未填写变量不会由 AI 猜测。
