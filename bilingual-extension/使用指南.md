# KOL 双语沟通助手 · 安装与使用指南

> 这是一个浏览器侧边栏小工具，帮你跟海外红人沟通。装一次就能用，话术和物料全团队共享、实时同步。
> 有问题先看本文最后的「常见问题」，还不行找管理员。

---

## 一、这是什么 / 能干嘛

- 自动把红人发来的外语消息，在下方翻成中文
- 一键分析对方意图、生成「中英双语」回复，复制就能发
- 回复不满意，让 AI 接着改，直到满意
- 主动发消息、砍价、催稿、催发布都有现成话术
- 像 ChatGPT 一样随时问 AI（翻译、思路、流程都行）
- logo、教程图、参考视频、链接，全团队一个地方共享

支持 **Chrome / Edge 浏览器**，用于网页版 **Instagram / Gmail / Outlook**。

---

## 二、安装（每人做一次，约 3 分钟）

> 开始前，先找管理员要两样东西：① 插件压缩包 `kol-bilingual-assistant.zip` ② **团队口令**。

1. 把压缩包**解压**到一个固定文件夹（比如桌面），**别留在压缩包里**。
2. 打开 Chrome，地址栏输入 `chrome://extensions` 回车（Edge 输入 `edge://extensions`）。
3. 打开右上角的「**开发者模式**」开关。
4. 点「**加载已解压的扩展程序**」，选中刚解压出来的那个文件夹（点进去能直接看到 `manifest.json` 文件的那一层）。
5. 点浏览器右上角的「拼图」图标，把「KOL 双语沟通助手」**固定**到工具栏（点一下图钉）。
6. 打开网页版 Instagram，点插件图标 → 右侧弹出侧边栏。
7. 点侧边栏顶部「**⚙️ 服务器设置**」：
   - **服务地址**：已经默认填好，不用改。
   - **团队口令**：填管理员给你的口令（注意大小写、前后别留空格）。
   - **管理员口令**：留空（普通成员不填）。
   - 点「**保存并连接**」，右上角显示「**千问已连接**」就成功了。

✅ 装好后：Instagram 私信里红人的外语消息，下方会自动出现中文翻译。

## 三、三个板块怎么用

侧边栏顶部有三个标签，按场景切换。第一次打开有「👋 新手指引」，看完点「知道了」收起；顶部「❓」可随时再看。

### 📥 红人来消息（对方发消息、你要回复时用）

1. 把红人的消息**粘贴**到「红人的消息」框（或在 IG 选中那条消息，再点「读取当前选中文字」）。
2. （可选）在「**你想怎么回复**」写一句你的思路，例如：`委婉拒绝` / `砍到 250` / `先稳住等 TL 确认`。不写也行，AI 会自己判断。
3. 点「**分析并建议回复**」，几秒后会给出：
   - 对方意图、合作阶段
   - **建议外语回复 + 中文对照**（点「复制」直接发给红人）
   - **内部处理建议**（紫色框）—— ⚠️ **只给你自己看的，别发给红人！** 比如要不要先问 TL、有什么风险。
4. **回复不满意？** 在「不满意？告诉 AI 怎么改」写一句怎么改（`太长了精简点` / `语气更热情` / `主语翻反了，是对方让我发`），点「**让 AI 改这条回复**」，可以反复改。
5. **想逐句核对翻译？** 点「📑 生成逐句对照」，一句外语配一句中文，一眼看出哪句不对。
6. 也可以点「📖 从话术库选回复话术」，直接挑现成话术 → 选语言 → 填变量 → 用。

### 📤 我主动发（你主动去联系红人时用）

三个入口任选：
- **入口一 · 从话术库选**：选场景（冷启动 / 催稿 / 砍价…）、填变量，点「生成这条话术」。
- **入口二 · 自由说一句**：用中文写你想表达的意思，点「润色并生成双语」。
- **入口三 · 话术库**：打开完整话术库挑。

顶部「**输出语言**」先选目标语言。生成后，点「**外语 → 中文回译核对**」确认意思没错再发。

### 💬 问 AI（啥都能问，像 ChatGPT）

多轮对话，遇到拿不准的随时问，例如：
- 「这句泰语什么意思？」
- 「怎么委婉地拒绝涨价？」
- 「帮我写一段西班牙语的合作介绍。」
- 「这个红人值不值得合作？」

输入框打字，**回车发送**；点「清空」重开一段。

## 四、话术库 与 物料库（顶部按钮）

### 📚 话术库
全团队共享的话术。可以**搜索**、点「**+ 新增**」手动加一条、点「**导入**」批量加。
（编辑、删除已有话术需要管理员。）

### 📎 物料库
存放 logo、教程图、参考视频、链接、文字说明，按产品分类。
- **看**：点「👁 查看」加载预览，再点「收起」。
- **用**：
  - 图片 / 视频 → 点「下载」存到本地，再发给红人（建议下载一次、之后从本地发）。
  - 链接 → 点「复制链接」直接粘进 DM。
  - 文字说明 → 点「复制文字」。
- **加**：点「+ 上传物料」，选类型（📷 图片 / 🎬 视频 / 🔗 链接 / 📝 文字说明）。**所有人都能加。**
- 💡 大视频建议存成「🔗 链接」（云盘 / YouTube），复制即用，比上传文件更快。

## 五、怎么更新

- **话术、AI 效果更新** → ✅ **全自动，你什么都不用做**（存在服务器，实时同步）。
- **插件界面有大改时**（偶尔）→ 管理员会发新压缩包，你解压覆盖原文件夹，再到 `chrome://extensions` 点插件卡片上的「🔄 刷新」即可。

> 服务器设置、口令只填一次，**关浏览器也不会丢**，下次自动连。

## 六、常见问题（FAQ）

| 现象 | 怎么办 |
|---|---|
| 右上角显示「待配置 / 口令不正确」 | 打开「⚙️ 服务器设置」，确认**团队口令**填对（大小写、别多空格），点保存 |
| 红人消息没自动翻译 | 确认右上角是「千问已连接」；**刷新一下 Instagram 页面** |
| 翻译 / 生成失败、超时 | 网络或服务器繁忙，稍等重试；一直不行找管理员 |
| 紫色「内部处理建议」能发给红人吗 | ❌ **绝对不能**，那是给你自己看的内部提醒 |
| 关了浏览器要重新填口令吗 | 不用，填一次永久保存 |

### 附：管理员维护（仅管理员看）

- 服务装在公司 VPS。更新服务端：VPS 终端运行 `bash ~/kol-update.sh`。
- 更换 API Key / 团队口令：`cd ~/kol-repo/bilingual-extension && bash vps/setup.sh`。
- 话术库备份：插件「话术库 → 导出备份」，或在 VPS 的 `~/kol-data/` 目录。
- 编辑 / 删除话术、删除物料：在「⚙️ 服务器设置」填入**管理员口令**后即可操作。
