# KOL 双语沟通助手 · 完整使用说明（手动安装版）

> 商店还没上线，先用「解压版」手动装。装一次，以后更新覆盖文件 + 点一下刷新即可。
> 一句话：它帮你**看懂红人外语消息、生成双语回复、出主意、整理合作进展、提醒别漏回红人**。

---

## 一、安装（手动装解压版，一次性）

1. **解压**：把拿到的 `kol-assistant-vX.X.X.zip` 解压到一个**固定文件夹**（比如桌面建个 `kol插件`）。⚠️ **这个文件夹别删、别移**，插件一直从这里运行。
2. **加载**：浏览器地址栏输入 `chrome://extensions` 回车 → 打开右上角「**开发者模式**」→ 点「**加载已解压的扩展程序**」→ 选刚才那个文件夹。
3. **钉出来**：浏览器右上角点 🧩 拼图图标 → 找到「KOL 双语沟通助手」→ 点图钉 📌，它就常驻工具栏了。
4. （每次开 Chrome 若弹「**要停用开发者模式扩展吗**」，点「**取消 / 保留**」即可。）

### 以后更新（10 秒）
1. 把新版 zip 解压后的文件，**覆盖进原来那个文件夹**（选「全部替换」）——
   ⚠️ 关键是**同一个文件夹、别换位置、别删插件**；
2. `chrome://extensions` → 找到本插件 → 点 **刷新按钮 ↻**（不要「移除」再重新加载）；
3. 回 Instagram 标签页按 **F5** 刷新。
> 只要替进的是同一个文件夹（路径不变）→ 插件 ID 不变 → 你的设置和数据**都不会丢**。
> 从 0.22.1 起插件 ID 已固定，就算不小心换了文件夹也不会丢；但仍建议用「覆盖同一文件夹 + 刷新」最稳。
> 双保险：⚙️ 服务器设置里有「💾导出备份 / 📂导入恢复」，换电脑或出意外时用它搬家 / 还原。

## 二、第一次设置（一次性，2 项）

点工具栏插件图标，右边弹出侧边栏：

1. **⚙️ 服务器设置**
   - 服务地址 + 团队口令：**找管理员要**；
   - 管理员口令：普通成员**留空**；
   - 点「保存并连接」，显示「已连接」就对了。

2. **🔔 提醒 → ⚙️ 身份设置**
   - 「我是哪个产品的」：选你负责的（recco / rythmix / aicatch / vivavideo）；
   - 「我的 IG 用户名」：一般自动认出，没有就手填（如 `recco_anna`）。

## 三、主界面：和红人沟通（就一列，没有切换 tab）

最大那个框：**粘红人的话**，或**直接问 AI**。然后按需点按钮：

### 💬 生成双语回复
- 点它 → **3-5 秒先出回复**：上面是**左右分屏**（左外语 | 右中文，逐句对照，**鼠标移到一行两边都高亮**），**外语那列可以直接点进去改**；
- 满意就点「**复制外语**」发给红人；
- 想改：下面「**想改？写一句**」里写「太长精简 / 语气热情点 / 别提价格」→「让 AI 改」，上面会更新；
- 再下面**折叠的「识别 & 内部提醒」**是分析（识别语言、言外之意、要不要问 TL 等），**晚几秒自动补上**，不挡你拿回复。
- ⚠️ 紫色「内部处理建议」是给你自己看的，**别发给红人**。
- 你也可以先在「你想怎么回复」里写一句意图（如「委婉拒绝」），再生成。

### ❓ 这是什么意思
看不懂红人那句话时点它 → AI **逐句讲清意思 + 言外之意**。不用管是翻译问题还是上下文问题，AI 自己拿捏。

### 🤔 这怎么办
拿不准怎么处理时点它（例：**这个人死活不同意怎么办？**）→ AI 结合对话给你**出主意**。

> **上下文 AI 会自动读**：上面三个按钮，背后都会自动读你**当前打开的那个 IG 对话**当参考，你啥都不用做。想更准，可以在「对话参考」里把对话粘进去。

### 📖 从话术库选回复话术
要现成话术时点它，从话术库里挑。

## 四、📋 当前合作情况（再也不怕忘了聊到哪一步）

**解决什么烦恼？**
红人一多，你经常忘：这个人到底聊到哪了？合同发没发？钱付没付？初稿来没来？每次都要翻半天聊天记录。

**你要做什么？——基本啥都不用做。**
你平时**正常和红人聊天、看对话**就行。插件在后台帮你记账，自动整理成一张「**进展清单**」。

**在哪看、看到啥？**
点「红人来消息」上面的「📋 当前合作情况」，里面是这样一张清单（✅=做了，⬜=没做/在等）：

```
✅ 报价确认 — 300USD 已谈妥
✅ 要到账号 — 已给 IG handle
⬜ 合同已签 — 未见
✅ 积分充值 — 已充
⬜ 初稿 — 在等
✅ 付款 — 已转
⚠️ 备注：红人最近想延期，留意
```
**一眼就知道走到哪、还差啥。**

**举个例子：** 你今天跟 Anna 聊了报价，明天又聊了合同。你不用记——你**切到别的对话时，它就自动更新这张清单**了。过几天你忘了"到底给 Anna 发合同没"，点开这张清单一看「⬜ 合同已签 — 未见」，哦，还没发。

**想立刻更新 / 它没自动更新时：** 点「📥 抓取整段」(把对话拉进来) 或自己粘贴 → 点「🔄 总结这段对话」。清单你也可以自己改、自己加一行。
**这单做完了：** 点「✅ 此合作已完结·不再提醒」，以后不再烦你。

## 五、🔔 提醒（别漏回红人、别忘了催）

**解决什么烦恼？**
红人几十上百个，谁发消息来你没回、谁答应给东西结果没给，太容易漏。这个帮你盯着。

**你要做什么？——也基本不用管。**
插件图标上会有个**红色数字**（=有几个待处理）。更省事的是：**你每天打开 Chrome，它会自动弹一个小窗口**，像 todo 清单一样告诉你今天该处理谁。

**它提醒你三种事：**
- **📥 待回复** = 红人发消息来了，你**还没回**。（就像微信小红点，但只挑"红人发的、你没回的"。你回了它就消。）
- **⏳ 待跟进** = 你问红人要**账号 / 合同 / 初稿**，他说"好"结果**一直没给** → 到点提醒你**去催**。
- **📝 待办** = 你自己记的事（比如"周五给 Anna 发新品"），到点提醒你。

**举个例子：** 你昨天问一个红人要 IG 账号，他回"ok"，但今天还没给你。明天你打开 Chrome，弹窗里就会出现「⏳ 待跟进：答应给 IG handle 但还没给」——提醒你去催他。

**每条提醒上能做：** 「打开对话」直接跳过去、「不用提醒了」划掉、群里的可以「🔕 这个群别再提醒」。

**两个顺手的小功能：**
- **🪄 智能加待办**：直接打一句话「**5天后小树枝交稿**」「**明天下午3点催 Anna**」，它自动帮你定好时间提醒。
- **会话里的小提示**：你跟红人聊合作、但**还没约交稿时间**时，IG 输入框上方会冒一条黄提示「还没约交稿时间」，点一下就能插一句话去问，或记进待办。

## 六、关于安全（放心用）

- 它**只读你自己打开的页面**，就像有人在旁边帮你看屏幕；
- **绝不**登录你的账号、**绝不**替你发消息 / 点赞 / 关注；
- 提醒和上下文也是读你正常刷到的内容，**不偷偷扫 IG、不碰 IG 接口**；
- 需要 AI 处理时，只把**那几句对话文本**发到**团队自己的服务器**。

## 七、常见问题

- **桌面通知不弹？** 多半是**系统挡了 Chrome 通知**或开了勿扰。Windows：设置→系统→通知→打开 Google Chrome；Mac：系统设置→通知→Chrome 允许。（IG 网页那个「通知」开关跟插件无关。）
- **翻译 / 回复不工作？** 看侧边栏右上角是不是「已连接」；不行就在 ⚙️ 服务器设置点一下「保存并连接」。
- **抓取要点很多次吗？** 不用。累积是后台自动的，「📥 抓取整段」点**一下**即可；日常你正常看对话、切走，进展会自动更新。
- **更新要卸载重装吗？** 不用。覆盖同一文件夹 → `chrome://extensions` 点刷新 ↻ → 刷新 IG。设置不丢。
- **以前装过别的版本？** 在 `chrome://extensions` 把旧的那个「已解压」**移除**，只留一个，免得跑两遍。
