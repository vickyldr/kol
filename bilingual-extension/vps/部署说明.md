# VPS 集中部署说明（腾讯云轻量 · Ubuntu）

把 AI 服务装到你的腾讯云 VPS 上，团队成员只需在插件里填一次「服务地址 + 团队口令」即可使用。
以后你改 AI 逻辑/话术，只更新这台 VPS，全团队立刻生效，他们什么都不用做。

- 你的公网 IP：`106.54.206.174`
- 端口：`3210`
- 安全：用「团队口令」防止别人盗用你的额度。第一版走 HTTP（不加密传输），
  内部小团队可接受；以后想加 HTTPS 再绑域名升级。

---

## 一、把服务文件传到 VPS

1. 腾讯云控制台 → 轻量应用服务器 → 你的实例 → 点「**登录**」，打开网页版终端（OrcaTerm）。
2. 在终端窗口找到「**上传文件**」按钮，把我发给你的 `kol-server.zip` 上传到主目录。
3. 在终端里依次运行（每行回车）：

   ```bash
   cd ~
   sudo apt-get update -y && sudo apt-get install -y unzip
   unzip -o kol-server.zip -d kol-server
   cd kol-server
   bash setup.sh
   ```

4. 按提示依次填：
   - **阿里云百炼 API Key**；
   - **团队口令**：每个同事插件里都填这个；
   - **管理员口令**：只有你自己填，用来编辑/删除话术（要和团队口令不同）。
   脚本会自动装 Node、设为开机自启、并做本机自检。

## 二、放行端口（最容易漏的一步）

腾讯云轻量自带防火墙，默认**不开** 3210 端口，不放行同事就连不上：

1. 控制台 → 轻量应用服务器 → 你的实例 → 「**防火墙**」标签。
2. 「添加规则」：应用类型自定义，协议 **TCP**，端口 **3210**，来源 `0.0.0.0/0`，确定。

## 三、验证

浏览器直接打开：`http://106.54.206.174:3210/health`
能看到一段 `{"ok":true,...}` 的文字，就说明外网通了。

## 四、团队成员怎么用

每位同事在插件里：

1. 打开侧边栏 → 顶部「**⚙️ 服务器设置**」。
2. 服务地址填：`http://106.54.206.174:3210`
3. 团队口令填：你设置的那个团队口令。
4. 管理员口令：**同事留空**（普通成员只能新增和查看）。
5. 点「保存并连接」，右上角显示「千问已连接」即可。

**你自己（管理员）**：服务地址、团队口令照填，外加在「管理员口令」里填你设的那个管理员口令——
填了之后，话术库里每条话术下面才会出现「编辑 / 删除」按钮。

> 本机自己单人用（不连 VPS）：服务地址保持 `http://127.0.0.1:3210`，口令全部留空即可。

## 五、日常维护

```bash
sudo systemctl restart kol-assistant      # 重启服务
sudo journalctl -u kol-assistant -f       # 实时看日志
```

- **更新 AI 逻辑/话术**：上传新的 `kol-server.zip`，重新 `unzip` 覆盖后
  `sudo systemctl restart kol-assistant` 即可，团队无需任何操作。
- **改 Key 或团队口令**：重新 `bash setup.sh` 即可。
